## Installation

1. Download the package to a local folder (e.g. ~/epe4md-py/) by running:
    ```
    git clone https://github.com/letfritz/epe4md-py.git
    ```
2. Run Python IDE and navigate to the folder (~/epe4md-py/), then run the main.py script.
